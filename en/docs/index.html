Key Features

Section titled “Key Features”

100% Free & Private

No installation required. Runs entirely in your browser. All processing happens locally, so your data and fonts remain private.

Advanced Font Styling

Apply solid colors, gradients, and image patterns. Add definition with strokes (outlines), depth with shadows, and control anti-aliasing with a powerful Sharpness setting.

Professional Typography

Fine-tune glyph metrics, line height, and configure kerning pairs for pixel-perfect text layout and readability.

Smart Texture Packing

Leverages an efficient Guillotine algorithm to auto-pack glyphs into the smallest texture atlas, or lets you define fixed, power-of-two dimensions for GPU optimization.

Broad Export Compatibility

Export in Text (.fnt/.txt), XML, and Binary formats compatible with popular game engines like Unity, Godot, and Unreal Engine.

Real-Time Preview

See your changes instantly. The live canvas provides immediate feedback on styling, metrics, and kerning adjustments.

Character & Glyph Management

Section titled “Character & Glyph Management”

Full Unicode Support

Includes full support for international scripts, emoji, and symbols, with IME compatibility for easy character input.

Flexible Font Import

Import TTF, OTF, and WOFF files. A robust fallback system lets you combine multiple fonts to expand character coverage.

Image Glyphs

Integrate custom icons, symbols, or logos directly into your font by mapping images to specific characters.

Automatic Deduplication

Smart character processing automatically removes duplicates, ensuring an efficient and clean character set.

Advanced Features

Section titled “Advanced Features”

Pixel-Perfect Control

Use the unique ‘Sharp’ feature to snap glyph vectors to the pixel grid, transforming smooth fonts into crisp, pixel-perfect art with a single slider.

Interactive Gradient Editor

Create linear and radial gradients with unlimited color stops. Visually edit colors, positions, and angles with instant feedback.

Visual Kerning

Visually adjust the spacing between character pairs. The tool provides highlighting and precise pixel-level control for professional results.

Project Management

Save and load projects in the native .sbf format. The tool also supports importing legacy .ltr files from Littera for easy migration.
