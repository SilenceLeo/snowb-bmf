## 核心功能

<CardGrid stagger>
	<Card title="100% 免费且保护隐私" icon="laptop">
		无需安装，完全在浏览器中运行。所有处理都在本地完成，确保您的数据和字体文件安全私密。
	</Card>
	<Card title="高级字体样式" icon="pen">
		支持纯色、渐变和图像填充。通过描边（轮廓）增强清晰度，利用阴影增加深度，并使用强大的“锐利度”设置控制抗锯齿效果。
	</Card>
	<Card title="专业排版控制" icon="pencil">
		微调字形度量、行高，并配置字偶距（Kerning），实现像素级精准的文本布局和卓越的可读性。
	</Card>
	<Card title="智能纹理打包" icon="puzzle">
		采用高效的“铡刀算法”（Guillotine）自动将字形打包到最小的纹理图集，或允许您定义固定的二次幂尺寸以优化 GPU 性能。
	</Card>
	<Card title="广泛的导出兼容性" icon="document">
		支持导出为文本（.fnt/.txt）、XML 和二进制格式，兼容 Unity、Godot、Unreal Engine 等主流游戏引擎。
	</Card>
	<Card title="实时预览" icon="magnifier">
		所有更改即时可见。实时画布提供对样式、度量和字偶距调整的即时反馈。
	</Card>
</CardGrid>

## 字符与字形管理

<CardGrid stagger>
	<Card title="完整的 Unicode 支持" icon="translate">
		全面支持国际化文字、表情符号和特殊符号，并兼容输入法（IME），方便字符输入。
	</Card>
	<Card title="灵活的字体导入" icon="add-document">
		支持导入 TTF、OTF 和 WOFF 格式的字体文件。强大的“回退字体”系统允许您组合多个字体以扩展字符覆盖范围。
	</Card>
	<Card title="图像字形" icon="seti:image">
		通过将图像映射到特定字符，将自定义图标、符号或 Logo 直接集成到您的字体中。
	</Card>
	<Card title="自动去重" icon="approve-check-circle">
		智能字符处理功能可自动移除重复项，确保字符集高效、整洁。
	</Card>
</CardGrid>

## 高级特性

<CardGrid stagger>
	<Card title="像素级精准控制" icon="puzzle">
		使用独特的“锐利度”（Sharp）功能将字形矢量对齐到像素网格，只需拖动一个滑块，即可将平滑的字体转换为清晰锐利的像素风格字体。
	</Card>
	<Card title="交互式渐变编辑器" icon="star">
		创建包含无限色标的线性和径向渐变。通过可视化操作即时编辑颜色、位置和角度。
	</Card>
	<Card title="可视化字偶距调整" icon="setting">
		直观地调整字符对之间的间距。工具提供高亮显示和精确的像素级控制，助您实现专业效果。
	</Card>
	<Card title="项目管理" icon="open-book">
		支持以原生的 .sbf 格式保存和加载项目。同时兼容导入来自 Littera 的旧版 .ltr 文件，轻松迁移。
	</Card>
</CardGrid>
