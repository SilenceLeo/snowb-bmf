## 核心功能

<CardGrid stagger>
	<Card title="100% 免费且保护隐私" icon="laptop">
		不用安装，浏览器里直接用。所有处理在本地跑，数据和字体文件不会离开你的设备。
	</Card>
	<Card title="高级字体样式" icon="pen">
		纯色、渐变、图像填充。描边让轮廓更清晰，阴影加立体感，”锐利度”控制抗锯齿效果。
	</Card>
	<Card title="专业排版控制" icon="pencil">
		调字形度量、行高和字偶距（Kerning），文本布局精确到像素。
	</Card>
	<Card title="智能纹理打包" icon="puzzle">
		Guillotine 算法自动把字形打包到最小纹理图集。也可以指定固定的二次幂尺寸，对 GPU 更友好。
	</Card>
	<Card title="广泛的导出兼容性" icon="document">
		6 种导出格式：文本（.fnt/.txt）、XML、二进制、JSON、C Header 和 MSDF Atlas JSON。兼容 Unity、Godot、Unreal Engine、嵌入式系统和 MSDF 着色器管线。
	</Card>
	<Card title="实时预览" icon="magnifier">
		改什么都能立刻看到效果，样式、度量、字偶距调整马上反映在画布上。
	</Card>
	<Card title="快捷键" icon="laptop">
		Space+拖拽平移、Ctrl+滚轮缩放等快捷键，让字体编辑更流畅。
	</Card>
	<Card title="PWA 与离线" icon="approve-check-circle">
		装成渐进式 Web 应用，离线也能干活，上线后自动检测更新。
	</Card>
</CardGrid>

## 字符与字形管理

<CardGrid stagger>
	<Card title="完整的 Unicode 支持" icon="translate">
		国际化文字、表情符号、特殊符号都支持，兼容输入法（IME）。
	</Card>
	<Card title="灵活的字体导入" icon="add-document">
		导入 TTF、OTF、WOFF 字体文件。”回退字体”能组合多个字体，扩展字符覆盖范围。
	</Card>
	<Card title="图像字形" icon="seti:image">
		把图像映射到字符，自定义图标、符号、Logo 都能塞进字体里。
	</Card>
	<Card title="自动去重" icon="approve-check-circle">
		自动移除重复字符，保持字符集整洁高效。
	</Card>
</CardGrid>

## 高级特性

<CardGrid stagger>
	<Card title="像素级精准控制" icon="puzzle">
		“锐利度”（Sharp）功能把字形矢量对齐到像素网格。拖一下滑块，平滑字体就变成清晰的像素风。
	</Card>
	<Card title="交互式渐变编辑器" icon="star">
		线性、径向渐变都能做，色标数量不限。颜色、位置、角度全部可视化编辑。
	</Card>
	<Card title="可视化字偶距调整" icon="setting">
		直观调字符对间距，有高亮显示，像素级精确。
	</Card>
	<Card title="SDF/MSDF 渲染" icon="star">
		用 msdfgen WASM 生成距离场字体，分辨率无关。SDF、PSDF、MSDF、MTSDF 四种模式，怎么缩放文本都清晰。
	</Card>
	<Card title="内阴影效果" icon="star">
		字形内部画阴影，做出雕刻和浮雕效果。跟外阴影叠加用更好看。
	</Card>
	<Card title="可变字体" icon="pen">
		自动检测变体轴，通过实时滑块调整字重、字宽、斜体及自定义轴。
	</Card>
	<Card title="有序网格打包" icon="puzzle">
		第三种打包模式，字形按行列整齐排列，列数可配置，布局规则可预测。
	</Card>
	<Card title="项目管理" icon="open-book">
		以 .sbf 格式保存和加载项目，兼容 Littera 旧版 .ltr 文件导入。
	</Card>
	<Card title="自动保存与持久化" icon="approve-check-circle">
		所有项目自动存到 IndexedDB。多项目工作区带选项卡，意外刷新也不怕丢东西。
	</Card>
</CardGrid>
